# AIML Hackathon 2026 - Student Resources

This public repository contains the necessary resources for the Passage Ranking Hackathon.

## Contents
- `INSTRUCTIONS.md`: Step-by-step student guide.
- `PROMPT.md`: Project context (required for LLM disclosure).
- `starter_notebook.ipynb`: A ready-to-use notebook to get you started.
- `requirements.txt`: Python dependencies.

## Quick Start
1. Clone this repo in your Kaggle Notebook.
2. Install requirements.
3. Open `starter_notebook.ipynb`.
